    String

    Todo tipo de caracteres, más utilizado para palabras o textos, pero incluso un número puede ser string, solo que pierde la capacidad de hacer operaiones.
    En Ruby se sabe que estas frente a uns string su lo vez entre comillas como estas "" o estas ''
    Un ejemplo de como definimos una variable string es "Hola"


    Array

    Arreglos de números, es decir un conjunto de valores guardados en una sola variable. Util en física para representar vectores y matrices. Tambien pueden guardarse cadenas de strings


    Symbol

    Muy confundido con un string, los simbolos tienen la caracteristica de ser inmutables, es decir, no pueden cambiar mediante metodos (por ejemplo no podemos hacer que un simbolo tenga todos sus caracteres en mayuscula con un upcase).
    Aparte su manejo en memoria es diferente, lo que lo vuelve más rapido para ser usado como claves de hashes que los strings. Mas información sobre la diferencia entre simbolos y strings
